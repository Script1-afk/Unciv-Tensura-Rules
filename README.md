# Unciv Tensura Rules Mod v1

Le mod **Unciv Tensura Rules** est conçu pour accompagner le mod Tensura et offrir une expérience de jeu équilibrée et thématique. Il modifie les règles de base d'Unciv pour refléter les mécaniques de l'univers de *Moi, quand je me réincarne en Slime*.

## Caractéristiques du Mod de Règles
- **Mécaniques de Jeu** :
  - **Évolution des Unités** : Les unités de monstres gagnent en puissance de combat (+10% par niveau d'évolution).
  - **Bonus de Science** : Après avoir débloqué la technologie 'Sagesse du Grand Sage', chaque citoyen génère +1 Science supplémentaire.
  - **Défense de la Foi** : Les merveilles mondiales fournissent un bonus substantiel de +5 Foi.

- **Conditions de Victoire Spécifiques** :
  - **Hégémonie de l'Octagramme** : Devenez le Seigneur Démon Suprême et contrôlez toutes les capitales originelles.
  - **Paix Mondiale de Jura** : Atteignez la victoire culturelle tout en possédant la merveille légendaire Grotte de Veldora.

- **Équilibrage Thématique** :
  - Les barbares sont remplacés par des 'Monstres Sauvages' plus redoutables (+20% de force).
  - Les unités magiques ignorent les pénalités de terrain, reflétant leur maîtrise de la magie.
  - Le bonheur global a un impact direct sur la production (bonus de 15%).

## Installation
1. Copiez l'URL de ce dépôt GitHub.
2. Dans Unciv, allez dans `Mods` > `Download mod from URL`.
3. Collez l'URL et cliquez sur `Download`.
4. Assurez-vous que ce mod de règles est activé en même temps que le mod **Unciv-Tensura**.

**Note** : Ce mod est **essentiel** pour que les mécaniques du mod Tensura fonctionnent comme prévu par le créateur.
